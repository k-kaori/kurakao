Webページに保存した入力情報やパスワードの消去方法（Microsoft Edge）

Microsoft EdgeのWebページに保存した入力情報やパスワードの消去方法をご案内します。

  • 一定の期間に保存したすべての情報を消去する
  • Webサイトごとにパスワードを消去する

一定の期間に保存したすべての情報を消去する

一定の期間に保存したパスワードや、以前入力したユーザー名やメールアドレスなどをすべて消去する場合は、次の手順を行います。

  1. Microsoft Edgeを起動する。
  2. 画面右上の[…]をクリックする。
  3. メニュー下の[設定]をクリックする。
  4. 設定のメニューをクリックし、[プライバシー、検索、サービス]をクリックする。
  5. 「閲覧データをクリア」の設定にある[クリアするデータの選択]をクリックする。
  6. 「時間の範囲」で「すべての期間」を選択し、[パスワード]と[オートフィル フォーム データ（フォームやカードを含む）]の項目をチェックし[今すぐクリア]をクリックする。 
    • 消去したいデータだけにチェックをつけてください。

以上で「一定の期間に保存したすべての情報を消去する」は完了です。

Webサイトごとにパスワードを消去する

保存したパスワードをウェブサイトごとに消去する場合は、次の手順を行います。

  1. Microsoft Edgeを起動する。
  2. 画面右上の[…]をクリックする。
  3. メニュー下の[設定]をクリックする。
  4. 設定の[プロファイル]をクリックする。
  5. 「プロファイル」の[パスワード]をクリックする。
  6. 「プロファイル/パスワード」の設定にある[保存されたパスワード]から、削除したいパスワードの右にある[…]をクリックして[削除]を選択する。
  7. 「パスワードを削除しました」と表示される。

以上で「Webサイトごとにパスワードを消去する」は完了です。
