Internet Explorer モードの設定方法（Microsoft Edge）

Microsoft Edge の「Internet Explorer モード」をセットアップする方法をご案内します。

  1. Microsoft Edgeを起動する。
  2. 画面右上の[…]をクリックする。
  3. メニュー下の[設定]をクリックする。
  4. 設定のメニューから[既定のブラウザー]をクリックする。
  5. 「Internet Explorer モードでサイトの再読み込みを許可」の設定を[許可]に変更します。
  6. 設定を変更すると、[再起動]を促すボタンが現れるのでクリックすると、Microsoft Edge が再起動し、設定の変更が反映されます。
  7. 「Internet Explorer モード」が設定されると、画面右上の［…］メニューに［Internet Explorerモードで再度読み込みする］が追加されます。
  8. 「Internet Explorer モードページ」の[追加]をクリックして「ページの追加」にURLを入力し、[追加]ボタンをクリックすると、Internet Explorer モードで自動的に開くページが登録されます。
  9. 追加されたURLは、Internet Explorer モードで開かれます。Edgeでの表示に戻すときは[Microsoft Edgeで開く]をクリックします。

以上で完了です。
