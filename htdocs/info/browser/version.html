インターネットブラウザのバージョンアップのお願い

ご利用のインターネットブラウザが古いバージョンの場合、インターネットブラウザを最新バージョンに変更することをおすすめします。

古いバージョンのインターネットブラウザをご利用の方へ

現在ご利用のインターネットブラウザは、BIGLOBEの各種ページ／サービスが正常に表示できない、サービスの機能が利用できない、セキュリティを保つことができないなど、予期せぬ不具合が発生する恐れがございます。

発生する可能性のある不具合（一例）

  • ページの表示内容が崩れてしまう
  • 古いページ内容が表示されてしまう（古いキャッシュが残りやすくなる）
  • コンテンツが正しく動作しない
  • 予期せぬ動作が実行される恐れがある
  • サポート（改善）が受けられず、データを盗まれたりなどセキュリティ上のリスクが高い

問題を解決するには

最新版のブラウザをインストールすることで、不具合を解決することができます。
最新版のインストールは、各ブラウザの「バージョンの確認方法」をご覧ください。

  • Microsoft Edge
  • Internet Explorer
  • Chrome
  • Firefox
  • Safari
