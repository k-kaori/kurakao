各種ログインページの暗号化技術「3DES」の脆弱性対応について

2019年4月22日 掲載
ビッグローブ株式会社
BIGLOBEカスタマーサポート

BIGLOBE会員のみなさま

平素よりBIGLOBEをご利用いただき、誠にありがとうございます。

インターネットの通信で使用される暗号化方式「3DES」では脆弱性が発見されており、悪意の第三者からの攻撃により通信の一部が第三者に漏えいする可能性があることが報告されています。
そのため、BIGLOBEでは、お客さまに安全・安心な環境でインターネットをご利用いただけるよう「3DES」の利用停止を進めており、2019年5月7日より各種BIGLOBEサービスへのログインでの利用を停止いたします。

この対応により、お客さまの環境によってはマイページなどのBIGLOBEサービスにログインできなくなる可能性があります。以下の対応方法をご確認のうえ、ご利用環境の更新をお願いいたします。

対応日時

2019年5月7日（火）

対応内容および症状

暗号化方式「3DES」の利用を停止いたします。
下記のOS・ブラウザをご利用のお客さまは、マイページ、各種手続き画面、Webメール、お問い合わせフォームなど『https://』から始まる各種ページにログインできなくなります。

影響のある利用環境

  • Windows XP Service Pack3 Internet Explorer 8 以前のバージョン
  • Android OS 4.3 以前のバージョン

対処方法

Windows XPをご利用のお客さま

最新のWindowsへの移行をご検討ください。
Windows XPは、2014年4月9日（日本時間）をもってサポート期間が終了しています。パソコンやインターネットを安全にご利用いただくため、お早めに最新版のWindows（Windows 10）への移行をご計画ください。
また、最新のWindowsに関する情報については「Microsoft Windows 製品サイト」をご確認ください。

Android OS 4.3以前のバージョンをご利用のお客さま

お使いのスマートフォンに標準搭載されているブラウザから、ChromeやFirefoxなど他社ブラウザへの移行をご検討ください。

BIGLOBEでは、より安心してご利用いただけるサービスを提供するため、今後も順次セキュリティ強化を進めてまいります。
今後とも、BIGLOBEをご愛顧賜りますよう、よろしくお願い申し上げます。
